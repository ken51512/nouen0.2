# 農園長者 MVP v0.5 — 自由研究統合テスト版

## 確定反映
- テストモード：全機能開放、8プランター、5日ごとの成長、収穫、販売、料理を一括生成
- 学校名・学年・クラス・名前を表紙に反映
- 初期設備と各費用、栽培野菜を自動集計
- 5日ごとの成長写真・コメントを自動配置
- 月次売上・経費・利益・お小遣い決算書
- 追加設備投資・プランター増設・ROI・回収期間
- 光合成、栽培条件、設備による改善、創意工夫
- 料理写真・レシピ・感想
- 栄養素と働き
- 父が作った専用アプリと連動した感想
- 次に育てる野菜と次回利益目標
- 活動の都度、自由研究コメントを促す導線

## テスト方法
「その他」→「テストモード」→「一括で全工程のサンプルを作る」→「自由研究レポートを見る」。
写真は実際の撮影時に手動追加してください。
