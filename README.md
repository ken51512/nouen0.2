# 農園長者 MVP v0.4（司令塔・記録・デバッグ・ダブルチェック強化版）

## 今回の検証改善

- 全身アバターを実装し、帽子・上着・ズボン・靴・アクセサリーをタップ時に即時反映
- 現在のプランター数・栽培作物・親設定単価・設備から収穫量、売上、原価、利益を予測
- プランター別の収支予測グラフ
- ROIと投資回収サイクル
- 設備投資前後の収支シミュレーション
- 野菜別の栄養素から、体のどこでどのような働きに関係するかを深掘り
- 下部5タブをまとめとして維持し、数字・野菜・設備・キャラクターから下位レイヤーへ遷移
- 確定仕様マスター、要求追跡表、自動検証結果を同梱
