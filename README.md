# 農園長者 MVP v0.6 RCテスト版

## 主な反映
- プレイヤー名を「○○社長」として全体表示
- 下部タブを「ホーム・農園・経営・図鑑・その他」に変更
- 着替えを「その他」へ移動
- 通知許可をチュートリアルに追加
- 水やり・5日ごとの観察ミッションと通知導線
- ホームに「今」「次」のミッション表示
- 収穫ボタン＋必須写真で収穫完了を確定
- 収穫後のみ「親に売る」を有効化
- 親モードに電気代・水道代（初期値各50円/プランター）
- 野菜図鑑から野菜別レシピへ遷移
- 経営タブに設備投資・回収期間・収支予測
- 「困ったときは？」Q&A

## テスト
