# 農園長者 MVP v0.3（9役検証改善版）

## 今回の中心改善
- 導入で「自分の努力でお小遣いを増やし、農園長者を目指す」目標を明示
- 下部5タブは各分野のまとめ・入口として維持
- キャラクター・野菜・数値・設備・文字をタップして階層的に深掘り
- 経営：売上、原価、利益、お金の流れ、設備投資、親ローン
- バッジ：カテゴリ、条件、進捗、取得日、報酬ポイント
- 設備投資：実際の購入価格、設置状態、投資前後比較
- ポイントは設備投資に使用不可
- 親ローンは無利息、利益からの返済割合は子どもが設定
- 着替えはタップした瞬間にキャラクターへ試着反映
- 栄養素と体への働きを野菜ごとに深掘り
- 自由研究はアプリ内プレビューと印刷画面を分離
- 全プランターは空・0%から開始し、実時間と観察記録に連動
- 既存機能を削除せず、樹形図型の階層で整理

## 9役確認
1. 企画：目的とゲームループ
2. 記録：確定仕様と仮値の分離
3. 実行：画面・状態・保存
4. UX：初回導線・現在地・戻る
5. 教育：食育・理科・英語・経営
6. 農業：実時間・設備・作物情報
7. デバッグ：重複報酬・状態不整合
8. PM：既存機能維持・段階実装
9. ダブルチェック：円・ポイント・設備・ローンの分離
