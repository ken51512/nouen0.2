# 農園長者 MVP v0.7.1 収穫・販売導線改善テスト版


## 公開確認
画面右上に **v0.7 TEST** と表示されれば最新版です。
旧画面が表示された場合でも、v0.7のService Workerは旧キャッシュを削除し、HTMLをネットワーク優先で取得します。

## 主要実装
- 下部タブ：ホーム／農園／経営／図鑑／その他
- 着替えは「その他」へ移動
- 入力名を「○○社長」として表示、社長Lvを採用
- 通知許可をチュートリアルで説明
- 水やり・5日ごとの観察ミッション、写真・コメント記録
- ホームに「今」「次」のミッション
- 記録達成状況とアプリ内生育率を連動
- 収穫ボタン、収穫写真、収穫完了状態
- 収穫後のみ「親に売る」を有効化
- 売上・経費・利益・お小遣いを自動集計
- 親モード：電気代・水道代 初期値各50円／プランター
- 経営タブ：追加設備、プランター増設、収益予測、ROI、回収期間
- 図鑑の野菜詳細からレシピを表示
- 料理写真・レシピ・感想を自由研究へ反映
- 困ったときは？ Q&A
- テストモード：全機能開放、時間進行、収穫・販売・料理・研究の検証

## GitHub Pagesへの配置
ZIPを展開し、中にあるファイルをリポジトリのルートへ上書きしてください。フォルダそのものを置かず、`index.html` がルート直下にある状態にします。


## v0.7.1修正
- 収穫前から「収穫ボタン」を常時表示し、未解放理由と現在の生育率を表示。
- 「親に売る」ボタンも常時表示し、収穫完了までロック理由を表示。
- 農園の「収穫」入口を、売上履歴ではなく収穫・販売対象一覧へ変更。
