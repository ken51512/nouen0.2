# 農園長者 MVP v0.2（検証改善版）

## Ver0.2の最優先改善
- 初回チュートリアル（目的→名前→キャラ→設備→開始手順）
- ホームに「農園長者への道」「今やること」
- 名前をホーム上部からタップ変更
- 全プランターは空・0%から開始
- 種まき後も0%スタート
- 生育状況は実際の種まき日からの日数に連動
- 「成長記録」を押しても勝手に成長しない
- 1日ごとの写真・コメントを成長記録として保存
- 全通常画面に下部タブを常時表示
- 詳細モーダルに戻るボタン
- 自由研究はアプリ内プレビューと印刷専用レイアウトを分離
- 既存機能を削除せず維持

## GitHub更新方法
リポジトリ `nouen-choja` のルートにある以下のファイルを、このZIP内の新しいファイルで上書きしてください。

- index.html
- manifest.json
- sw.js
- icon-192.png
- icon-512.png
- README.md

GitHub PagesのURLはそのままです。更新後、Safariに古い表示が残る場合はページ再読み込み、またはホーム画面アイコンを一度削除して追加し直してください。
